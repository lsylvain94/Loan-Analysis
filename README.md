# Loan-Analysis
Analyzing loan performance and risk using Power BI dashboards

Using Power BI, I created an interactive dashboard to explore loan status, funding amounts, and key financial indicators.

---

## 🎯 Objective
To analyze loan performance and identify patterns that can help financial institutions reduce risk and improve decision-making.

---

## 🛠️ Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

---

## 📈 Key Insights
- Certain loan categories show higher default risk
- Loan status varies significantly by borrower characteristics
- Funding amounts and repayment trends highlight risk patterns
